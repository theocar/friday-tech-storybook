"React StoryBook" à la rescousse !!


getstorybook.io

C'est quoi ?

Un outil qui permet d'isoler les composants d'interface de manière à pouvoir les tester plus facilement.

Plusieurs avantages

  • Accélère le développement front en facilitant les tests
  • Améliore la qualité du code en nous obligeant à découpler nos composants
  • Offre une documentation claire et solide
  • Facilite la communication avec les Designer et les QA

Comment ça marche ?

  • React Storybook se lance sur une url dédiée (Port 9001)
  • On Crée des "stories" à partir du code source de notre projet
  • On indique à React Storybook où trouver les "stories" de manière à pouvoir les exploiter

    • Qu'est ce qu'une "story"

  • Représente un état figé d'un composant (exemple: "loading", "loaded", "success"...)
  • Techniquement c'est juste une fonction qui renvoi un ReactElement

    • Webpack Hot Module Replacement

    Cette technologie permet de mettre à jour le code javascript sans recharger la page
